## Interactive States

### 1. Initial State (No Video)
```
Chat Button: 💬 (no pulse)
Chat Panel: Disabled
Welcome Message: "Upload and analyze a video first"
```

### 2. Video Processing
```
Status: ⚙️ "Processing video.mp4..."
Chat Button: 💬 (no pulse)
Chat Panel: Disabled
```

### 3. Video Ready
```
Status: ✅ "Analysis complete!"
Chat Button: 💬 with GREEN PULSING DOT
Chat Panel: Enabled
Welcome Message: "📹 Video Ready! Ask anything!"
Suggested Questions: Loaded
```

### 4. Active Conversation
```
Chat Button: 💬 (no pulse when panel open)
Messages: User (red) + Bot (white)
Typing: ● ● ● when bot thinking
Auto-scroll: To latest message
```

### 5. Timestamp Hover
```
Normal: [00:01:45] blue gradient
Hover: [00:01:45] scaled 105%, stronger shadow
```

### 6. Suggested Question Hover
```
Normal: Gray background, left-aligned
Hover: Light blue bg, slides 4px right
```

## Animation Timings

**Panel Slide:** 0.3s cubic-bezier(0.4, 0, 0.2, 1)
**Message Appear:** 0.3s ease-out
**Button Hover:** 0.2s ease
**Typing Dots:** 1.4s infinite (staggered 0.2s)
**Notification Pulse:** 2s infinite
